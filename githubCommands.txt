INITIALIZING A REPO
---------------------------------------------------------------
git init
git add .
git status
git commit -m "Commit Name"
git branch -M "master"
git remote add origin https://github.com/Matchujk/.git --- you can omit this if you cloned the repo and are pushing an existing commit
git push origin master


CLONE A REPO
---------------------------------------------------------------
git clone https://github.com/Matchujk/.git


PUSHING EDITED FILE TO REPO
---------------------------------------------------------------
git add .
git branch -M "master"
git commit -m "commit name"
git push origin master
